## install go

````shell
export GOROOT=/usr/local/go
export GOPATH=$HOME/your/go/code
````

### install dev tools

````sh
go get -u github.com/ddollar/forego
go get -u github.com/ddollar/rerun
````

## start

````sh
forego start
````
